# Agent guidelines

When working in this repo:

- **Commits MUST use [Conventional Commits](https://www.conventionalcommits.org/).** Every commit message must match `<type>(<scope>): <description>` or at minimum `<type>: <description>` (e.g. `feat:`, `fix:`, `docs:`, `chore:`). Non-conventional messages are not acceptable for this repository.
- **NEVER disclose PII in commits.** Do not add, log, or commit names, emails, IDs, credentials, or any other personally identifiable information. Keep secrets and env-specific values in `.env` or local config only.
