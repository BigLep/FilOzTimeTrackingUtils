# Agent guidelines

When working in this repo:

- **ALWAYS use [Conventional Commits](https://www.conventionalcommits.org/)** (e.g. `feat:`, `fix:`, `docs:`, `chore:`). Format: `<type>(<scope>): <description>`.
- **NEVER disclose PII in commits.** Do not add, log, or commit names, emails, IDs, credentials, or any other personally identifiable information. Keep secrets and env-specific values in `.env` or local config only.
